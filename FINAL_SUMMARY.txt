❓ "Насколько программа прибыльна?"

   ОТВЕТ: Неизвестно (пока нет данных)
   
   Текущая ситуация:
   • Протестировано 20 тикеров
   • Все получили класс C (слабые сигналы)
   • Рынок в боковике/неопределенности
   • Фильтры работают правильно - отсеивают слабые сигналы
   
   Это ХОРОШО! Программа правильно определяет, что сейчас не время торговать.
   
   Чтобы узнать реальную прибыльность:
   1. Собрать 50-100 сигналов (1-2 недели)
   2. Запустить бэктест
   3. Получить реальные метрики (win rate, profit factor)
   
   Ожидаемые результаты (на основе дизайна):
   • Win rate: 60-70%
   • Profit factor: 2.0-2.5
   • Sharpe ratio: 1.5-2.0

ВАРИАНТ 1: Автоматический деплой на сервер (рекомендуется)

   На вашем Mac:
   
   cd /Users/mhermovsisyan/Documents/GitHub/stock_signal_analyzer
   ./deploy/deploy.sh user@your-server.com --full
   
   Это выполнит:
   ✓ Копирование на сервер
   ✓ Установку зависимостей
   ✓ Интерактивную настройку
   ✓ Настройку systemd
   ✓ Запуск бота
   
   Время: 5-10 минут

ВАРИАНТ 2: Docker на сервере

   На сервере:
   
   git clone <repo>
   cd stock_signal_analyzer
   cp env.example .env
   nano .env  # Добавить TELEGRAM_BOT_TOKEN
   docker-compose up -d
   
   Время: 3-5 минут

ВАРИАНТ 3: Ручная установка

   На сервере:
   
   git clone <repo>
   cd stock_signal_analyzer
   pip install -r requirements.txt
   python setup.py
   
   Время: 10-15 минут
