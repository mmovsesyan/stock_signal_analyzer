# 🤖 Обновление меню Telegram бота

**Дата:** 2026-05-01  
**Версия:** 1.3.0

---

## 🎯 Что изменилось

Добавлено полное управление автосбором сигналов через меню Telegram бота.

### Новый раздел: ⚙️ Настройки

Раздел "Уведомления" переименован в "Настройки" и расширен:

```
⚙️ Настройки
├── 🤖 Настройка автосбора
│   ├── ✅/❌ Дефолтные тикеры (30)
│   ├── ➕ Добавить свои тикеры
│   ├── 📋 Показать текущие
│   └── 🗑️ Очистить свои тикеры
└── 🔔 Уведомления
    ├── 🔔 Уведомления ВКЛ
    └── 🔕 Уведомления ВЫКЛ
```

---

## 📋 Новые возможности

### 1. Управление дефолтными тикерами

**Кнопка:** `✅/❌ Дефолтные тикеры (30)`

- Включает/выключает 30 дефолтных тикеров для автосбора
- Дефолтные тикеры: 10 РФ + 10 US + 10 дивидендных
- По умолчанию: включены

### 2. Добавление своих тикеров

**Кнопка:** `➕ Добавить свои тикеры`

- Позволяет добавить собственные тикеры для автосбора
- Вводятся через пробел: `AAPL TSLA SBER.ME GAZP.ME`
- Автоматически нормализуются и дедуплицируются

### 3. Просмотр конфигурации

**Кнопка:** `📋 Показать текущие`

Показывает полную конфигурацию:
- Статус дефолтных тикеров (вкл/выкл)
- Список ваших добавленных тикеров
- Ваш watchlist
- Итоговое количество тикеров для автосбора

### 4. Очистка своих тикеров

**Кнопка:** `🗑️ Очистить свои тикеры`

- Удаляет все добавленные вами тикеры
- Watchlist и дефолтные тикеры не затрагиваются

## 🔧 Технические изменения

### Файл: `stock_signal_analyzer/user_store.py`

Добавлены новые поля в `UserPrefs`:

```python
@dataclass
class UserPrefs:
    watchlist: list[str] = field(default_factory=list)
    notify_strong_outside: bool = True
    strong_threshold: float = 0.35
    notify_cooldown_sec: int = 86_400
    last_notify_ts: dict[str, float] = field(default_factory=dict)
    
    # НОВЫЕ ПОЛЯ
    autocollect_tickers: list[str] = field(default_factory=list)
    use_default_tickers: bool = True
```

### Файл: `telegram_bot.py`

**Новые функции клавиатур:**
- `_settings_menu_keyboard()` - главное меню настроек
- `_autocollect_menu_keyboard(uid)` - меню автосбора (динамическое)
- Обновлена `_notify_menu_keyboard()` - кнопка "Назад в настройки"

**Новые обработчики:**
- `on_menu_section_settings()` - открыть настройки
- `on_menu_autocollect()` - открыть настройку автосбора
- `on_menu_toggle_default_tickers()` - переключить дефолтные тикеры
- `on_menu_add_custom_tickers()` - добавить свои тикеры
- `on_menu_show_custom_tickers()` - показать конфигурацию
- `on_menu_clear_custom_tickers()` - очистить свои тикеры
- `on_menu_back_to_settings()` - вернуться в настройки

**Обновлённые функции:**
- `_collect_tickers_for_user(uid)` - теперь учитывает:
  1. Watchlist пользователя
  2. Пользовательские тикеры (`autocollect_tickers`)
  3. Дефолтные 30 тикеров (если `use_default_tickers=True`)
  
- `autocollect_job()` - собирает тикеры от всех пользователей через `_collect_tickers_for_user()`

- `on_pending_text()` - добавлена обработка `add_custom` для ввода тикеров

## 📖 Использование

### Пример 1: Отключить дефолтные тикеры

```
1. Открыть: ⚙️ Настройки
2. Нажать: 🤖 Настройка автосбора
3. Нажать: ✅ Дефолтные тикеры (30)
4. Статус изменится на: ❌ Дефолтные тикеры (30)
```

Теперь автосбор будет использовать только ваш watchlist и добавленные тикеры.

### Пример 2: Добавить свои тикеры

```
1. Открыть: ⚙️ Настройки
2. Нажать: 🤖 Настройка автосбора
3. Нажать: ➕ Добавить свои тикеры
4. Ввести: AAPL TSLA SBER.ME GAZP.ME
5. Готово! Тикеры добавлены
```

### Пример 3: Проверить конфигурацию

```
1. Открыть: ⚙️ Настройки
2. Нажать: 🤖 Настройка автосбора
3. Нажать: 📋 Показать текущие
```

Вы увидите:
```
📋 Текущая конфигурация автосбора

Дефолтные тикеры (30): ✅ включены

Ваши тикеры (4):
AAPL, TSLA, SBER.ME, GAZP.ME

Watchlist (2):
NVDA, GOOGL

Итого тикеров для автосбора: 36
```

## 🎯 Логика автосбора

Автосбор (`autocollect_job`) собирает тикеры в следующем порядке:

1. **Watchlist** - всегда включён
2. **Пользовательские тикеры** (`autocollect_tickers`) - всегда включены
3. **Дефолтные 30 тикеров** - если `use_default_tickers=True`

Все тикеры дедуплицируются (уникальные).

### Дефолтные 30 тикеров

**РФ голубые (10):**
- SBER.ME, GAZP.ME, LKOH.ME, GMKN.ME, ROSN.ME
- NVTK.ME, TATN.ME, MOEX.ME, MGNT.ME, CHMF.ME

**US голубые (10):**
- AAPL, MSFT, GOOGL, AMZN, NVDA
- META, TSLA, JPM, JNJ, V

**Дивидендные (10):**
- KO, PEP, XOM, PFE, T
- VZ, INTC, BAC, WMT, DIS
