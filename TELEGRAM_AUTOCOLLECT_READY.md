## 🔧 Возможности

### 1. Переключение дефолтных тикеров
**Кнопка:** `✅/❌ Дефолтные тикеры (30)`

- Одним нажатием включить/выключить 30 дефолтных тикеров
- Дефолтные: 10 РФ + 10 US + 10 дивидендных
- Статус отображается на кнопке (✅ или ❌)

### 2. Добавление своих тикеров
**Кнопка:** `➕ Добавить свои тикеры`

- Вводите тикеры через пробел: `AAPL TSLA SBER.ME`
- Автоматическая нормализация и дедупликация
- Сохраняются отдельно от watchlist

### 3. Просмотр конфигурации
**Кнопка:** `📋 Показать текущие`

Показывает:
- Статус дефолтных тикеров (вкл/выкл)
- Список ваших добавленных тикеров
- Ваш watchlist
- **Итоговое количество тикеров для автосбора**

### 4. Очистка своих тикеров
**Кнопка:** `🗑️ Очистить свои тикеры`

- Удаляет только добавленные вами тикеры
- Watchlist и дефолтные не затрагиваются

## 🎮 Как использовать

### Сценарий 1: Использовать только свои тикеры

```
1. Открыть: ⚙️ Настройки
2. Нажать: 🤖 Настройка автосбора
3. Нажать: ✅ Дефолтные тикеры (30)
   → Статус изменится на: ❌ Дефолтные тикеры (30)
4. Нажать: ➕ Добавить свои тикеры
5. Ввести: AAPL MSFT GOOGL SBER.ME GAZP.ME
6. Готово!
```

Теперь автосбор будет использовать только ваши 5 тикеров + watchlist.

### Сценарий 2: Добавить к дефолтным свои тикеры

```
1. Открыть: ⚙️ Настройки
2. Нажать: 🤖 Настройка автосбора
3. Нажать: ➕ Добавить свои тикеры
4. Ввести: TSLA NVDA PLTR
5. Готово!
```

Теперь автосбор будет использовать: 30 дефолтных + 3 ваших + watchlist.

### Сценарий 3: Проверить, что будет собираться

```
1. Открыть: ⚙️ Настройки
2. Нажать: 🤖 Настройка автосбора
3. Нажать: 📋 Показать текущие
```

Вы увидите полную конфигурацию:
```
📋 Текущая конфигурация автосбора

Дефолтные тикеры (30): ✅ включены

Ваши тикеры (3):
TSLA, NVDA, PLTR

Watchlist (2):
AAPL, GOOGL

Итого тикеров для автосбора: 35
```

## 🔄 Логика автосбора

Автосбор собирает тикеры в следующем порядке:

1. **Watchlist** - всегда включён
2. **Ваши тикеры** (`autocollect_tickers`) - всегда включены
3. **Дефолтные 30** - если `use_default_tickers=True`

Все тикеры дедуплицируются (уникальные).

### Дефолтные 30 тикеров

**РФ голубые (10):**
```
SBER.ME, GAZP.ME, LKOH.ME, GMKN.ME, ROSN.ME
NVTK.ME, TATN.ME, MOEX.ME, MGNT.ME, CHMF.ME
```

**US голубые (10):**
```
AAPL, MSFT, GOOGL, AMZN, NVDA
META, TSLA, JPM, JNJ, V
```

**Дивидендные (10):**
```
KO, PEP, XOM, PFE, T
VZ, INTC, BAC, WMT, DIS
```

## 💾 Хранение данных

Все настройки сохраняются в `telegram_users.json`:

```json
{
  "users": {
    "123456789": {
      "watchlist": ["AAPL", "GOOGL"],
      "notify_strong_outside": true,
      "strong_threshold": 0.35,
      "notify_cooldown_sec": 86400,
      "last_notify_ts": {},
      "autocollect_tickers": ["TSLA", "NVDA", "PLTR"],
      "use_default_tickers": true
    }
  }
}
```
