### 🤖 Настройка автосбора

```
✅ Дефолтные тикеры (30)
или
❌ Дефолтные тикеры (30)

➕ Добавить свои тикеры    📋 Показать текущие
🗑️ Очистить свои тикеры
⬅️ Назад в настройки       🏠 Главное меню
```

**Описание:**
```
🤖 Настройка автосбора

Дефолтные тикеры (30): включены
Ваши тикеры: 0

Автосбор использует:
• Дефолтные 30 тикеров (если включены)
• Ваши добавленные тикеры
• Ваш watchlist
```

**При добавлении тикеров:**
```
➕ Добавить тикеры для автосбора

Введите тикеры через пробел, например:
AAPL TSLA SBER.ME GAZP.ME

Или напишите: отмена
```

**При просмотре конфигурации:**
```
📋 Текущая конфигурация автосбора

Дефолтные тикеры (30): ✅ включены

Ваши тикеры (4):
AAPL, TSLA, SBER.ME, GAZP.ME

Watchlist (2):
NVDA, GOOGL

Итого тикеров для автосбора: 36
```
